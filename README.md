# Pinisi

Sebuah eksperimen sosial

=======

## How to Install

Please follow these steps:

1. `git clone`
2. Create virtualenv
3. `pip install -r requirements.txt`

## How to Run

To run the program, do:

1. From shell, run in the virtual environment (in Linux: `source env/bin/activate`)
2. `python run.py`
3. or `foreman start`
